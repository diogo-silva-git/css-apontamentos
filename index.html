Universal Selector

  • Styles.css : * {color: orange;}

    • Select elements of every type (as in the whole document)

  • Index.html.css :<p>example<p>

Type Selector

  • Styles.css : div {color: rgb(116, 96, 228);}

    • Selects all elements of the given element type, syntax is just the name of the element.

  • Index.html : <div>example<div>

Class Selector

  • Styles.css : .example {color: red;}
